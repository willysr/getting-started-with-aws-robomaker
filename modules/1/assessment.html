    Assessment for Module 1 - Introduction to AWS RoboMaker

    Linux commands

    This assignment is to help you self-assess your understanding on AWS RoboMaker’s architecture. Read carefully each question. Once answered, you will receive feedback. Good luck!

    Q1

    AWS RoboMaker integrates which of the following software and services to speed up the development of robotics application? Select all that apply:

    * Cloud Extensions for ROS
    * Development Environment
    * Simulation
    * Fleet management

    Q2

    Gazebo is a robot simulation software.

    1. True.
    2. False.

    Q3

    The cost for development in AWS Robomaker depends on what components? Select all the is True.

    * visualization service
    * time for running the services
    * development environment
    * the storage used

    Q4

    Match the pairs.

    0. Development Environment 0. IDE that interfaces with ROS
    1. Amazon S3 bucket 1. Storage in the cloud
    2. Rviz 2. Data visualization
    3. Gazebo 3. Simulation
    4. Fleet Management 4. Deployment in robot
